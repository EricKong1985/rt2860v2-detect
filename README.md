
## 首先下载openwrt的编译环境

下载openwrt的编译环境

``` git

git clone https://github.com/lixuande/openwrt-icbbox

```

探针的驱动层源码

```
cd openwrt-icbbox/package/
git clone https://github.com/lixuande/rt2860v2-detect

```

探针应用层源码

```
cd openwrt-icbbox/package/
git clone https://github.com/lixuande/rt2860v2-detect-user

```
